# Discord Token Generator

Uses Python and httpx library to create discord accounts


## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install required modules.

```bash
pip install -r requirements.txt
```


## Usage
On Windows:
```bash
python main.py
```
On Mac:
```bash
python3 main.py
```
